i-Ball's Emotions

Play around with i-Ball who uses a trained classifier to guess whether you're happy or sad from the way you look.

Download Project Launch Codelab

Project Guide

  • Step 1: Download the project. If you have not trained an image project that can recognize emotions, you can use this data and train a new model. Use the categories "happy" and "sad". Try to use your own pictures to train the model as well.

  • Step 2: After training your model, write down your API key and the model name to use in the starter project you downloaded.
